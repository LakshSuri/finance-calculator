# Finance Calculator

A Python-based financial analysis tool that models investment growth, helps plan financial goals, stores calculation history, and visualizes different investment scenarios.

I built this project to practice Python programming while learning more about finance, investing concepts, and software development practices.

## Features

* Calculates future investment value using compound interest
* Calculates growth with monthly contributions
* Calculates the monthly amount needed to reach a financial goal
* Stores and displays calculation history
* Provides automated tests for calculator functions
* Creates investment growth charts
* Creates yearly growth bar charts
* Compares different investment return scenarios

Clone the repository and run:

```bash

```

This project uses automated tests to verify that the financial calculations continue working correctly.

Run tests with:

```bash

```
