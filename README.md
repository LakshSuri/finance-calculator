Finance Calculator:

This is a Python project that calculates how investments can grow over time.

I built this project to practice Python and learn more about finance and investing concepts.

What it does:
- Calculates future investment value
- Calculates growth with monthly contributions
- Uses compound interest calculations
- Shows results with formatted currency values

Built with:
- Python
- Git
- GitHub

How to run:
```bash
python3 main.py
```
